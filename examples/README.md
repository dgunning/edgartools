# EdgarTools Examples

Python scripts demonstrating EdgarTools functionality. These are production-ready scripts you can copy and adapt for your projects.

## How to Use

1. Install EdgarTools: `pip install edgartools`
2. Copy any script
3. Run: `python script_name.py`
