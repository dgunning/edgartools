# EdgarTools Domain Knowledge

This folder contains **committable** domain knowledge that helps future development.

## Structure

- **sec-forms/** - SEC form specifications, structures, and parsing research
- **xbrl/** - XBRL parsing knowledge, architecture, value presentation
- **patterns/** - Common patterns and solutions for recurring problems
- **architecture/** - System design documentation

## Guidelines

Content here should:
1. Help future developers (human or AI) understand the domain
2. Document decisions and their rationale
3. Be reusable across multiple development sessions
4. Not contain sensitive business information
